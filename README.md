
##  Check Email Address In Subdomain 

Mail-Sub-Check-hunt is a  Node.js tools helps with querying Check if email address is Subdomain.

Output Explanation
```
E= email
 D= domain info
  S= info.subdomain / info.subdomain
   TLD= tld info
```
